# Optimización de la Gestión Académica (Frontend)

Este repositorio contiene la interfaz de usuario (Frontend) para la plataforma de gestión académica del colegio , desarrollada con **React**.

## 🚀 Tecnologías Utilizadas
- **React.js** (con Vite / Create React App)
- **Tailwind CSS** / **CSS Modules** (para los estilos)
- **Axios** (para consumir la API del Backend)
